## About The Project

A Java desktop application using Swing framework to simulate the "King of Tokyo" board game.

Here's why:
* Semester project for CECS 343 - Intro to Software Engineering
* Practice Java desktop programming with Intellij IDE
* Be involved in the SDLC of the application with 3 other group members
* Manage working in a team settings, getting requirements in on time.

### Built With
* Java
* Swing UI
